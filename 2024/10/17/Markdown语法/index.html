Markdown语法

Markdown语法

标题

#一级标题
##二级标题
###三级标题
####四级标题
#####五级标题
######六级标题

强调

*斜体* 斜体
**粗体** 粗体
***粗斜体*** 粗斜体
~~删除线~~ 删除线

引用

>引用的话
>aaa

引用的话
aaa

列表

有序列表

1. 第一项（注意“1.”后面有空格）
2. 第二项
3. 第三项

显示效果：

  1. 第一项
  2. 第二项
  3. 第三项

无序列表

- 第一项（注意“-”后面有空格）
- 第二项
- 第三项

显示效果：

  • 第一项
  • 第二项
  • 第三项

代码语法

```代码语言（写上代码才会有高亮）
反引号包裹代码块
```

`printf()`  一对反引号编写行内代码
效果：printf()

分隔线

-–  （三个连续的减号）

链接

超链接语法代码

[超链接显示名](超链接地址 “超链接title”)
如：[zwcsm](https://zwcsm.github.io/ “zwcsm的博客”)
效果：zwcsm

尖括号

使用尖括号可以很方便地把URL或者email地址变成可点击的链接
如：<https://zwcsm.github.io/>
效果：https://zwcsm.github.io/

带格式化链接

如：*[zwcsm](https://zwcsm.github.io/ “zwcsm的博客”)*
效果：**zwcsm**

图片

图片语法代码

![图片alt文本](图片地址 “图片title”)
如：![zwcsm](img.png “zwcsm的头像”)
效果：

链接图片

给图片增加链接，将图像的 Markdown 括在方括号中，然后将链接添加在圆括号中。

![图片alt文本](图片地址 “图片title”)
如：[![zwcsm](./Markdown语法/img.png “zwcsm的头像”)](https://zwcsm.github.io/)
效果：

转义字符

要显示原本用于格式化 Markdown 文档的字符，在字符前面添加反斜杠字符 \ 。

常见的有  \（反斜杠）  *（星号）  `（反引号）

表格

使用三个或多个（—）创建每列的标题，并使用（|）分隔每列
在标题行中的连字符的左侧，右侧或两侧添加冒号（:），将列中的文本对齐

| | 姓名 | 性别 | 年龄 |
| :————– | :—: | —: |
| 张三 | 男 | 22 |
| 李四 | 女 | 20 |

姓名 性别 年龄
张三 男 22
李四 女 20
